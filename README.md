## Retail Sales Performance & Customer Behaviour Analysis (SQL)

## Project Overview

This project focuses on analyzing retail store sales data using SQL.  
A structured relational database was designed to simulate real-world retail operations including customers, products, orders, and order details.

The goal of this project was to extract meaningful business insights from transactional data using optimized SQL queries.

---

## Database Structure

The database includes the following tables:

- `customers`
- `orders`
- `order_details`
- `products`

Primary and foreign key relationships were implemented to ensure data integrity and proper relational mapping.

---

## Business Problems Solved

This project answers key business questions such as:

- Who is the top spending customer?
- Which product sells the most?
- What is the monthly revenue trend?
- Who are the repeat customers?
- What is the average order value?
- Which product category generates the highest revenue?
- What is the revenue contribution per customer?
- Which month generated the highest revenue by category?
